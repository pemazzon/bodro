# bodro, bodrobold and dobodro
The BOx DRAwing figlet/toilet font (DRO for pronounce pourposes)

It comes in 3 variant:

  * bodro: normal

  * bodrobold: bold

  * dobodro: DOuble line BOx DRAwing
